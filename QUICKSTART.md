# Быстрый старт

1. Создайте файл `.env` с токеном бота:
   ```
   BOT_TOKEN=your_bot_token_here
   ```

2. Запустите бота:
   ```bash
   docker-compose up -d
   ```

3. Проверьте логи:
   ```bash
   docker-compose logs -f
   ```

4. Откройте бота в Telegram и отправьте `/start`

## Первое использование

1. Добавьте поставщика: `/suppliers` → "➕ Добавить поставщика"
2. Сгенерируйте заказ: `/generate` → выберите поставщика → загрузите три файла

## Формат файлов

- **Прайс-лист** - Excel файл с прайс-листом производителя
- **Заказ на склад** - Excel файл, структура соответствует прайс-листу
- **Предзаказы клиентов** - Excel файл со списком предзаказов

Поддерживаются форматы `.xls` и `.xlsx`.
